Apostrophe

Īsumā: Apostrophe ir jQuery spraudnis, kas padara iespējamu interneta lapās rakstīt latviski, ja datorā šādas iespējas nav.

Tas strādā tāpat, kā tad, kad ieinstalēts daudziem zināmais "Tildes birojs" - ar apostrofa taustiņu ('). Piemēram, nospiežot ' un burtu a, teksta ievades laukā tiek ierakstīts burts ā un tā tālāk...


Vēlies rakstīt latviski jebkurā interneta vietnē? - Izmanto bookmarklet’u!
A' - “ievelc” šo bookmarklet’u saišu jeb linku joslā un tad, kad gribēsi kādā interneta vietnē rakstīt latviski, vienkārši noklikšķini uz tā!

Starp citu, ja tev ir iespēja, nemokies un uzinstalē latviešu valodas tastatūras izkārtojumu savā datorā: laacz.lv/apostrofs!

Izstrādātājiem

Lai arī tavas interneta vietnes apmeklētāji varētu rakstīt latviski pat tad, ja datorā nav instalēta latviešu valodas taustiņu izkārtojuma, izmanto šo jQuery spraudni!

Apostrophe pieejams Github’ā: https://github.com/taai/Apostrophe

Lai aktivizētu spraudni konkrētam ievades laukam:

{% highlight javascript %} $('#ievades_lauka_id').apostrophe(); {% endhighlight %}

Lai lai deaktivizētu spraudni konkrētam ievades laukam:

{% highlight javascript %} $('#ievades_lauka_id').apostrophe(false); {% endhighlight %}

Lai aktivizētu spraudni visiem lapā esošajiem ievades laukiem:

{% highlight javascript %} $('textarea, input[type=text]').apostrophe(); {% endhighlight %}

Saderība: jQuery 1.2 vai jaunāks

Pārlūkprogrammas: Firefox, Safari, Chrome, Internet Explorer 6+, Opera
